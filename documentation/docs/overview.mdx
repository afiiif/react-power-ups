# Overview

React Power-Ups is a collection of React hooks designed to make common tasks in React development easier and more efficient. While the hooks themselves are **simple**, their purpose is to **save time and effort** for developers. **Rather than having to write the same hooks repeatedly** across multiple projects, developers can easily import the hooks they need from this package and get to work right away.

This package was built using **TypeScript**, which provides type-checking and improves the overall development experience. TypeScript catches errors early on and provides **autocomplete** and **documentation**, making it easier to maintain and understand the codebase.

## Key Features

⚖️ **Lightweight**: Only 6.3 kB (minified size)

🌳 **Tree-Shakeable**: Remove unused code from your production builds

🛡 **TypeScript**: Ensuring type safety and easy integration with your TypeScript projects

🌐 **SSR support**: You can use it with confidence in your server-rendered React applications

📱 **Cross-Platform**: Can be used in **React Native** too

💡 **Real-World Examples**: Our documentation includes real-world examples of how to use our hooks in various scenarios, helping you get up and running quickly and easily

export const HookList = ({ items }) => (
  <ul className="list-none pl-0">
    {items.map(({ name, href, desc, usecase }) => {
      return (
        <li key={name} className="mb-5">
          <a
            href={`/docs/${href}`}
            className="block hover:no-underline shadow-md border border-solid border-[--ifm-color-gray-200] rounded-lg hover:border-[--ifm-color-primary] relative left-0 hover:left-3 transition-all overflow-hidden group"
          >
            <h4 className="p-4 pb-0 mb-1 text-lg text-[--ifm-color-primary-darker]">{name}</h4>
            <div
              className="px-4 pb-3 text-[--ifm-font-color-base]"
              dangerouslySetInnerHTML={{ __html: desc }}
            />
            {usecase && (
              <p className="mb-0 px-4 py-2 text-[--ifm-color-gray-700] bg-[--ifm-color-gray-100] text-sm group-hover:bg-[--ifm-color-primary] group-hover:text-[--ifm-color-secondary] dark:group-hover:text-[--ifm-color-gray-800]">
                💡 {usecase}
              </p>
            )}
          </a>
        </li>
      );
    })}
  </ul>
);
